## üöÄ Quick Start

1. **Clone the repo**  
   ```bash
   git clone https://github.com/flatlogic/ecommerce-frontend.git
   cd ecommerce-frontend
   ```
2. **Install dependencies**  
   ```bash
   yarn install  # Or npm install
   ```
3. **Run the app**  
   ```bash
   yarn start
   ```
   Navigate to http://localhost:3000/.

4. **Build for production**  
   ```bash
   yarn build
   ```
